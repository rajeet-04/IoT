<!-- Key contracts from Phase 1 backend that this frontend consumes. -->

From Phase 1 backend (Plan 01-02):
```javascript
// POST /api/auth/signup — {email, password} -> 201 {user: {id, email}} + Set-Cookie
// POST /api/auth/login — {email, password} -> 200 {user: {id, email}} + Set-Cookie
// POST /api/auth/logout — {} -> 200 {message: "Logged out"} + Clear-Cookie
// GET /api/auth/me — -> 200 {user: {id, email, createdAt}} or 401
// POST /api/auth/forgot-password — {email} -> 200 {message: "..."}
// POST /api/auth/reset-password — {token, newPassword} -> 200 {message: "..."}
// All cookies: httpOnly, secure (production), sameSite: 'lax'
```

From Phase 1 device API (Plan 01-03):
```javascript
// GET /api/devices — -> 200 {devices: [{id, deviceId, name, status, lastSeen}]}
// POST /api/devices — {name?} -> 201 {device: {id, deviceId, name, token}}
// PUT /api/devices/:id — {name} -> 200 {device: {...}}
// DELETE /api/devices/:id — -> 200 {message: "..."}
// All require authentication via httpOnly cookie
```

Contracts this plan creates:
```typescript
// frontend/src/lib/api.ts exports:
export async function apiFetch(path: string, options?: RequestInit): Promise<Response>
// Prefixes with NEXT_PUBLIC_BACKEND_URL, includes credentials: 'include' for cookies
// Throws on non-2xx responses with parsed error body

// frontend/src/lib/auth.ts exports:
export async function login(email: string, password: string): Promise<{user: User}>
export async function register(email: string, password: string): Promise<{user: User}>
export async function logout(): Promise<void>
export async function getCurrentUser(): Promise<User | null>
// User type: { id: string, email: string, createdAt: string }

// frontend/src/app/dashboard/layout.tsx:

```

       - const BACKEND_URL = process.env.NEXT_PUBLIC_BACKEND_URL || 'http://localhost:3000'
       - Export async function apiFetch(path: string, options: RequestInit = {}):
         - const url = `${BACKEND_URL}${path}`
         - const response = await fetch(url, { ...options, credentials: 'include', headers: { 'Content-Type': 'application/json', ...options.headers } })
         - If !response.ok: const error = await response.json().catch(() => ({ error: response.statusText })); throw new Error(error.error || response.statusText)
         - Return response
       - Helper functions: apiGet(path), apiPost(path, body), apiPut(path, body), apiDelete(path) — each calls apiFetch with appropriate method

       - Export async function logout(): await apiPost('/api/auth/logout', {})
