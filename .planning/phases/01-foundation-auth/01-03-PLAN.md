  <action>
    1. Create src/routes/device.routes.js:
       - Import express, zod, Device model, requireAuth middleware, generateDeviceToken
       - Define zod schemas:
         - createDeviceSchema: z.object({ body: z.object({ name: z.string().min(1).max(50).optional() }) })
         - updateDeviceSchema: z.object({ body: z.object({ name: z.string().min(1).max(50) }) })
       - All routes protected by requireAuth middleware
       - POST /: validate with createDeviceSchema, generate deviceToken via generateDeviceToken(), create Device with { userId: req.userId, deviceId: deviceToken, name: body.name }, return 201 { device: { id: device._id, deviceId: device.deviceId, name: device.name, token: deviceToken } } — the token is returned ONCE at creation for user to flash to ESP32
       - GET /: find all Devices where userId === req.userId, return 200 { devices: devices.map(d => ({ id: d._id, deviceId: d.deviceId, name: d.name, status: d.status, lastSeen: d.lastSeen })) }
       - GET /:id: find Device where _id === req.params.id AND userId === req.userId, return 404 if not found, return 200 { device: { id, deviceId, name, status, lastSeen, createdAt } }
       - PUT /:id: validate with updateDeviceSchema, findOneAndUpdate where _id === req.params.id AND userId === req.userId, return 404 if not found, return 200 { device: updated device }
       - DELETE /:id: findOneAndDelete where _id === req.params.id AND userId === req.userId, return 404 if not found, return 200 { message: 'Device deleted successfully' }
       - Export router
    2. Update src/index.js to mount device routes:
       - Import deviceRoutes from './routes/device.routes.js'
       - Add: app.use('/api/devices', deviceRoutes)
