  <acceptance_criteria>
    - src/ws/hub.js exports attachWebSocketHub(server, app) function
    - attachWebSocketHub creates WebSocket.Server with { server, path: '/ws' }
    - On connection: extracts "Authorization" header from upgrade request
    - Validates header format: must be "Bearer <token>" where token is non-empty string
    - If no/invalid auth header: ws.close(4001, 'Unauthorized') and return
    - If auth header present: calls Device.findOne({ deviceId: token }) — token IS the deviceId (UUID from Phase 1)
    - If device not found in DB: ws.close(4003, 'Device not registered') and return
    - If device found: calls registry.register(deviceId, ws), sends {"type":"welcome","deviceId":"<id>"} to client
    - Server sends ping every 30 seconds via setInterval(ws.ping(), 30000)
    - On pong timeout (no pong within 10s of ping): ws.terminate() and registry.unregister(deviceId)
    - On ws 'message' event: calls routeMessage(deviceId, message, registry)
    - On ws 'close' event: calls registry.unregister(deviceId), clears ping interval
    - On ws 'error' event: logs error, calls registry.unregister(deviceId)
    - src/index.js updated to import and call attachWebSocketHub(server, app) after app.listen()
    - Graceful shutdown: on SIGTERM, broadcasts close message to all connections, closes ws server, then proceeds with existing shutdown
  </acceptance_criteria>
  <action>
    1. Create src/ws/hub.js:
       - Import { WebSocketServer } from 'ws', Device model, ConnectionRegistry, routeMessage
       - Export function attachWebSocketHub(server, app):
         - const registry = new ConnectionRegistry()
         - const wss = new WebSocketServer({ server, path: '/ws', verifyClient: (info, cb) => { cb(true) } }) — accept all at HTTP level, auth at WS level
         - wss.on('connection', async (ws, req) => {
           - Extract auth header: const authHeader = req.headers['authorization']
           - If !authHeader || !authHeader.startsWith('Bearer '): ws.close(4001, 'Unauthorized'); return
           - const token = authHeader.slice(7)
           - const device = await Device.findOne({ deviceId: token })
           - If !device: ws.close(4003, 'Device not registered'); return
           - const deviceId = device.deviceId
           - await registry.register(deviceId, ws)
           - ws.send(JSON.stringify({ type: 'welcome', deviceId }))
           - Set up ping interval: const pingInterval = setInterval(() => { if (ws.readyState === 1) { ws.ping(); ws._pongTimeout = setTimeout(() => ws.terminate(), 10000) } }, 30000)
           - ws.on('pong', () => { clearTimeout(ws._pongTimeout) })
           - ws.on('message', async (data) => { try { await routeMessage(deviceId, data.toString(), registry) } catch (err) { console.error('[WS] Message routing error:', err) } })
           - ws.on('close', async () => { clearInterval(pingInterval); clearTimeout(ws._pongTimeout); await registry.unregister(deviceId) })
           - ws.on('error', async (err) => { console.error('[WS] Error:', err.message); clearInterval(pingInterval); await registry.unregister(deviceId) })
         })
         - Export wss and registry for use in other modules (e.g., command routing in Phase 3)
         - Return { wss, registry }
    2. Update src/index.js:
       - Import { attachWebSocketHub } from './ws/hub.js'
       - After app.listen(), store server reference: const server = app.listen(...)
       - Call: const { wss, registry } = attachWebSocketHub(server, app)
       - Update gracefulShutdown to include: wss.close() before server.close()
       - Broadcast disconnect message before closing: for each ws in registry.getAll().values(), if ws.readyState === 1, ws.send(JSON.stringify({ type: 'server_shutdown', message: 'Server is restarting' }))
  </action>
