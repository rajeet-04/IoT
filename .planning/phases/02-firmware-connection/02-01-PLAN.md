Purpose: Provide the device connectivity layer that enables all real-time control and monitoring. Without this, the ESP32 cannot communicate with the backend.
Output: PlatformIO project with WiFi manager, WSS client, authentication, keepalive, and reconnection logic.

class WiFiManager {

  bool isConnected();                // Returns WiFi.status() == WL_CONNECTED
