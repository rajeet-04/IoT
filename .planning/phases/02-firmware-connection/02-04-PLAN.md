  </read_first>
  <acceptance_criteria>
    - src/ws/presenceMonitor.js exports PresenceMonitor class
    - Constructor accepts ConnectionRegistry instance and options object
    - Default options: staleThresholdMs = 90000 (90 seconds), checkIntervalMs = 30000 (30 seconds)
    - start() method sets up setInterval that runs checkStaleConnections() every checkIntervalMs
    - checkStaleConnections() queries Device.find({ lastSeen: { $lt: new Date(Date.now() - staleThresholdMs) }, status: 'online' })
    - For each stale device: updates status to 'offline', calls registry.unregister(deviceId) if still registered, logs warning
    - stop() method clears the interval
    - getStaleDevices() returns array of deviceId strings for devices past stale threshold
    - Monitor does NOT throw errors — all DB operations wrapped in try/catch
  </acceptance_criteria>
  <action>
    1. Create src/ws/presenceMonitor.js:
       - Import Device model from '../models/Device.js'
       - Export class PresenceMonitor:
         - constructor(registry, options = {}):
           - this._registry = registry
           - this._staleThresholdMs = options.staleThresholdMs || 90000
           - this._checkIntervalMs = options.checkIntervalMs || 30000
           - this._interval = null
         - start():
           - If this._interval already set, return (idempotent)
           - this._interval = setInterval(() => this._checkStaleConnections(), this._checkIntervalMs)
           - console.log(`[Presence] Monitor started (stale: ${this._staleThresholdMs}ms, check: ${this._checkIntervalMs}ms)`)
         - stop():
           - If this._interval, clearInterval(this._interval), this._interval = null
           - console.log('[Presence] Monitor stopped')
         - async _checkStaleConnections():
           - const threshold = new Date(Date.now() - this._staleThresholdMs)
           - const staleDevices = await Device.find({ lastSeen: { $lt: threshold }, status: 'online' })
           - For each device in staleDevices:
             - console.warn(`[Presence] Device ${device.deviceId} stale (last seen: ${device.lastSeen})`)
             - await Device.updateOne({ deviceId: device.deviceId }, { $set: { status: 'offline' } })
             - this._registry.unregister(device.deviceId) — removes from registry if still present
         - async getStaleDevices():
           - const threshold = new Date(Date.now() - this._staleThresholdMs)
           - const devices = await Device.find({ lastSeen: { $lt: threshold }, status: 'online' })
           - return devices.map(d => d.deviceId)
  </action>
