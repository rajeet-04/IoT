class RelayController {
public:
  RelayController(int gpioPin);

  <action>
    1. Create firmware/src/relay_controller.h:
       - #pragma once
       - Include <Arduino.h>, <ArduinoJson.h>
       - Class RelayController with private members: _gpioPin (int), _state (bool, default false), _deviceToken (String, passed in constructor or set via setter)
       - Public methods: constructor(gpioPin), begin(), setOn(), setOff(), setState(bool on), getState() returns bool, getStateJson() returns JsonDocument, setDeviceToken(token)
    2. Create firmware/src/relay_controller.cpp:
       - constructor(gpioPin): stores _gpioPin, sets _state = false
       - begin(): pinMode(_gpioPin, OUTPUT), digitalWrite(_gpioPin, LOW) — safe state is OFF, _state = false, Serial.println("Relay controller initialized (safe state: OFF)")

       - setOff(): digitalWrite(_gpioPin, LOW), _state = false, Serial.println("[RELAY] OFF")
       - setState(bool on): if on call setOn() else call setOff()
       - getState(): return _state
       - getStateJson(): creates StaticJsonDocument<128> doc, sets doc["type"] = "state_report", doc["relay"] = _state, doc["deviceId"] = _deviceToken, doc["uptime"] = millis(), returns doc
       - setDeviceToken(token): stores _deviceToken = token
    3. Wiring documentation (add as comment in relay_controller.h):
       - ESP32 GPIO 26 → transistor base (2N2222) via 1kΩ resistor
       - Transistor collector → relay coil negative terminal
       - Relay coil positive terminal → 5V power supply
       - Transistor emitter → GND
       - Flyback diode (1N4007): cathode to 5V side of coil, anode to transistor collector side
       - ESP32 GND → power supply GND (common ground required)
       - Add 100µF electrolytic capacitor between 5V and GND near relay module
